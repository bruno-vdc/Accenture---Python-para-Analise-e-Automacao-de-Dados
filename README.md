# Accenture - Python para Analise e Automacao de Dados

Repositório criado para a entrega dos 3 projetos desse Bootcamp da DIO.
Os projetos são:

  - **NotebookLM**: criação de um notebook temático na plataforma da Google para estudo de um assunto financeiro;
  - **Modelo de fraude**: aplicação de modelos de machine learning para identificar transações fraudulentas em uma base de dados real;
  - **Chatbot de IA**: criação de um assistente financeiro de IA com base em dados mockados.
